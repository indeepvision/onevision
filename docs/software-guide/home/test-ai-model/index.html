Test AI Model

In the Home section you can manually make a Brain inference. In the left menu you have the following actions:

  • Inference: Executes a Brain inference on the current image. This action is available when a Brain is loaded.
  • Switch Brain: Enables selection of different Brains for inference. This action is available when multiple Brains are loaded.
note

The manual inference of a Brain creates a capture in the History menu.
