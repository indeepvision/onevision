Actions Panel

The Side Panel is a collapsible container that houses tabs, each offering specific functionality depending on whether you are in the Production View or the 2D Display View.

Actions​

Within the Production View, the Side Panel only includes the Actions tab. This tab provides a suite of tools designed to streamline and expand the interaction capabilities within the Production View.

info

Examples of actions include a counter to track production quantities, a serial number identifier for production batches, or a toggle button to activate a flag for saving images. For detailed guidance on using these actions effectively, refer to the Actions section.
