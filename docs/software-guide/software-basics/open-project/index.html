Open a Project

Project files are managed internally by the application, they cannot be opened directly from the file system as you would do with a document of a text processor or a spreadsheet.

All the project files that you have saved are listed in the Open project window, which you can open from the Project menu. All projects are identified by a unique name and a Last edited date. You can also view some other details about the project in the right-side panel, as the author and the version of OneVision that was used to create the project.

To open a project, select it from the list and click Open, or double-click on it.

Clone project​

To clone one of the existing projects, click on the settings button next to the project and select Clone. It will require you to enter a new name for the project.

Import a project​

To import a project, click on the Import button in the bottom of the popup. It will open a file selection dialog where you can select the project file to import. The imported project will be added to the list of projects.

Retargeting​

When the project that is imported has been created with an older version of OneVision, before importing the project you will be asked to retarget it to the current version. This is necessary because the project file format may have changed between versions, and the application needs to know how to convert the old format to the new one.

In case of very old versions of the software it might not be possible to retarget the project. In this case, you will need to open the project with the version of OneVision that was used to create it, and recover the information manually, for example copying the program sequences.
