Add a Brain

Brains are the way that OneVision executes AI models. The same AI model can be used in more than one Brain.

To add a Brain, go to the Brains subsection in the Project section and click the + button in the top right corner. It will show a dialog which will ask for the Brain name. Name the Brain, click Create and save the changes. A new tab will be added for the Brain in the Brains section.
