Create and Run a Script

Let's start writing some code. We want to create a program that opens images from a folder and executes our brain on them. To do this we will create a program sequence.

Go to the Project section and then click on Programming. In the Sequences screen, click on the add button in the top left corner. This will open a new sequence popup, where you can enter the name of the sequence (for example test_sequence) and also select the type of sequence, in this case we will select SIMPLE.

tip

Sequence types

  • SIMPLE: It is executed once when the program starts and then it is finished.
  • CYCLIC: It is executed continuously, once finished it is executed again.
  • INIT: It is executed once when the program starts, before any other sequence execute.
  • END: It is executed once when the program is stopped, after other sequences have ended.

Now add the following lines of code:

Quickstart runtime sequence

// OneVision quickstart program

// Configure location of the images
const var IMAGES_FOLDER = "C:\dry_fruits_images\";

// List the images in the folder
var images_list = File.listImagesFromDir(IMAGES_FOLDER);

for (var i = 0; i < images_list.length; i++) {
    // Create the complete path to the image
    var image_filepath = IMAGES_FOLDER + images_list[i];

    // Load the image into an Image vision element
    var loaded_image = Image.fromEncodedFile(image_filepath);

    // Resize the image to match the input size
    loaded_image = loaded_image.resize(1224, 1024);

    // Execute inference on the image
    var inference_result = Brains.test_brain.inference(loaded_image);

    // Change the color of the segmentations
    inference_result.almonds.setColor("#ff0000");
    inference_result.hazelnuts.setColor("#00ff00");
    inference_result.peanuts.setColor("#0000ff");
    inference_result.pistachios.setColor("#00ffff");
    inference_result.raisins.setColor("#ff00ff");

    // Create a capture to send to the GUI
    var capture = new Capture("Dry fruits", loaded_image);
    capture.addVisionElement("almonds", inference_result.almonds);
    capture.addVisionElement("hazelnuts", inference_result.hazelnuts);
    capture.addVisionElement("peanuts", inference_result.peanuts);
    capture.addVisionElement("pistachios", inference_result.pistachios);
    capture.addVisionElement("raisins", inference_result.raisins);

    // Send the capture
    Visu.display.sendCapture(capture);
    Visu.update();

    // Wait some time
    waitTime(1000);
}

This code will open the images in the configured folder and execute inference on them.

Save the changes made to the project by clicking on Save in the left menu.

Run the program​

The program (or runtime) is executed in the Home section. Click on Start program to start the program. You will see how the images are opened and inference is executed on them. For each image a new capture is shown in the GUI.

After all the images have been processed and the for loop is finished, the program will stop automatically.

You can also stop the program manually by clicking on Stop program.
