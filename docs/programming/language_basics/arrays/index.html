Lists

In OneVision Scripts, lists are used to store collections of elements. They are intended to hold elements of the same type, but you could also store multiple types using the any type . Here’s how you define lists in OneVision:

List Declaration​

You can declare a list by specifying the type of elements it will hold followed by []:

// list of numbers
var numbers: number[] = [];

// list of strings
var fruits: string[] = [];

// list of mixed types
var mixedList: any[] = [1, "Hello", true];

List Initialization​

Lists can be initialized with elements enclosed in square brackets []:

var numbers: number[] = [1, 2, 3, 4, 5];
var fruits: string[] = ["Apple", "Banana", "Cherry"];

As it was explained, OneVision can infer the type in most cases when the initial value has a unique type: 

// List of numbers
var numbers = [1, 2, 3, 4, 5]; // variable numbers is initialized with number[] type

// List of strings
var fruits = ["Apple", "Banana", "Cherry"]; // variable fruits is initialized with string[] type

Accessing list Elements​

You can access elements of an list using their zero-based index:

var fruits: string[] = ["Apple", "Banana", "Cherry"];

fruits[0]; // Output: "Apple"
fruits[1]; // Output: "Banana"
fruits[2]; // Output: "Cherry"

Nested lists​

Nested lists, also known as multidimensional lists, are lists that contain other lists as their elements. These are useful for representing complex data structures like matrices, grids, or any data that naturally fits into a table-like format.

You can easily define nested lists as: 

var matrix: number[][]= [[1,2,3], [4,5,6], [7,8,9]]; 
var value = matrix[1][2];  // value = 6

List Methods and Properties​

Lists in OneVision are very similar to Typescript and come with built-in methods and properties for manipulating and accessing their elements. Some common methods include push, pop, splice, concat, etc.

List Properties​

PropertyDescriptionReturned Type
lengthReturns or sets the number of elements in an list.number

Mutator Methods​

MethodDescriptionReturned Type
push(item: T): numberAdds one element to the end of the list and returns the new length.number
pop(): T \| undefinedRemoves the last element from the list and returns it, or undefined if the list is empty.T \| undefined
shift(): T \| undefinedRemoves the first element from the list and returns it, or undefined if the list is empty.T \| undefined
unshift(item: T): numberAdds one element to the beginning of the list and returns the new length.number
splice(start: number, deleteCount: number): T[]Changes the contents of a list by removing existing elements returning the deleted elementsT[]
sort(): thisSorts the elements of a list in place and returns the list.this
reverse(): T[]Reverses the order of the elements in the list in place and returns the list.T[]

Accessor Methods​

MethodDescriptionReturned Type
concat(items: T[]): T[]Returns a new list that is a shallow copy of the original list with additional elements concatenated.T[]
includes(searchElement: T, fromIndex?: number): booleanDetermines whether a list includes a certain element, returning true or false as appropriate.boolean
slice(start: number, end?: number): T[]Returns a shallow copy of a portion of a list into a new list object selected from start to end (end not included).T[]
toString(): stringReturns a string representing the specified list and its elements.string
